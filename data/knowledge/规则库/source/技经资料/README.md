# 技经资料规则库

本目录包含技经资料审查相关的规则。

## 规则分类

### 01-字段抽取

关键字段提取和识别相关的规则，包括：
- 项目基本信息提取
- 投资金额识别
- 关键日期提取
- 责任人信息识别

### 02-一致性校验

技经资料内部和跨文档一致性校验规则，包括：
- 投资金额一致性
- 项目信息一致性
- 时间节点一致性
- 审批流程完整性

## 规则命名规范

规则 ID 格式：`R-JJ-{类别代码}-{序号}`

- `R-JJ-FE-XXX`：字段抽取类规则
- `R-JJ-CC-XXX`：一致性校验类规则

## 编写新规则

1. 确定规则分类，选择对应的子目录
2. 根据规则复杂度选择模板（参见 `templates/README.md`）
3. 复制模板并按规范填写
4. 更新本 README 的规则清单
5. 更新 `source/index.json`
