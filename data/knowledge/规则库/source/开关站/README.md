# 开关站规则库

本目录包含 10kV 开关站专项的审查规则。

## 规则分类

### 01-设备选型

设备选型相关的规则，包括：
- 变压器容量校验
- 开关柜型号规范
- 电缆截面选择
- 设备参数匹配

### 02-安全距离

安全距离和间距相关的规则，包括：
- 设备间距要求
- 电缆沟封堵
- 防火分区
- 安全净距

### 03-造价核查

造价和工程量相关的规则，包括：
- 材料单价合理性
- 工程量一致性
- 概算准确性
- 清册完整性

## 规则命名规范

规则 ID 格式：`R-KGZ-{类别代码}-{序号}`

- `R-KGZ-EQ-XXX`：设备选型类规则
- `R-KGZ-SF-XXX`：安全距离类规则
- `R-KGZ-CS-XXX`：造价核查类规则

## 编写新规则

1. 确定规则分类，选择对应的子目录
2. 根据规则复杂度选择模板（参见 `templates/README.md`）
3. 复制模板并按规范填写
4. 更新本 README 的规则清单
5. 更新 `source/index.json`
