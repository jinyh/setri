## 规则描述

校验可研报告、初设报告、概算书、材料清册中的工程量数据是否一致。确保各阶段文档中的工程量数据保持一致性，避免因数据不一致导致的造价偏差和施工问题。

## 适用范围

- **项目类型**：10kV 开关站
- **文件类型**：可研报告、初设报告、概算书、材料清册
- **检查阶段**：评审

## 规范依据

- **主依据**：20kV及以下配电网工程建设预算编制与计算规定（2022版）
- **管理依据**：国网上电司发展〔2026〕94号 业扩配套电网基建项目管理工作的意见

## 检查条件

### 输入文件

- **可研报告**：PDF 格式，必需
- **初设报告**：PDF 格式，必需
- **概算书**：Excel/PDF 格式，必需
- **材料清册**：Excel/PDF 格式，必需

### 检查维度

1. **电缆长度一致性**
   - 可研报告中的电缆路径长度
   - 初设报告中的电缆敷设长度
   - 概算书中的电缆工程量
   - 材料清册中的电缆采购量
   - 容差标准：±5%（考虑路径优化）

2. **设备数量一致性**
   - 变压器数量
   - 开关柜数量
   - 电缆终端数量
   - 容差标准：必须完全一致

3. **土建工程量一致性**
   - 开关站建筑面积
   - 电缆沟长度
   - 基础开挖量
   - 容差标准：±3%

### 判断逻辑（LLM 推理）

```
任务：提取并比对多份文档中的工程量数据

步骤：
1. 从可研报告中提取关键工程量（电缆长度、设备数量、土建量）
2. 从初设报告中提取相同项目的工程量
3. 从概算书中提取工程量清单
4. 从材料清册中提取材料数量
5. 比对各文档中相同项目的数值
6. 识别差异超过容差标准的项目
7. 分析差异原因（合理调整 vs 数据错误）
8. 生成一致性校验报告

容差标准：
- 电缆长度：±5%（考虑路径优化）
- 设备数量：必须完全一致
- 土建工程量：±3%
```

## 结果模板

### 不一致输出

**严重级别**：`major`

**问题描述**：
```
工程量数据不一致。{item_name} 在不同文档中的数值存在差异：
- 可研报告：{value_feasibility}
- 初设报告：{value_design}
- 概算书：{value_budget}
- 材料清册：{value_materials}

最大偏差：{max_deviation}%，超过允许容差 {tolerance}%。
```

**证据定位**：
- 可研报告：第 {page_feasibility} 页
- 初设报告：第 {page_design} 页
- 概算书：第 {page_budget} 页，工程量清单第 {row_budget} 行
- 材料清册：第 {sheet_materials} 表，第 {row_materials} 行

**规范依据**：20kV及以下配电网工程建设预算编制与计算规定（2022版）

**建议措施**：请核对各文档中的 {item_name} 数据，确保一致性。如存在合理调整，需在初设报告中说明调整原因。

## 例外情况

- 可研到初设的合理优化调整（需在初设报告中说明）
- 材料清册含损耗系数（需在清册中标注）
- 分期实施导致的数量差异（需在报告中说明分期方案）

## 执行提示

此规则需要 LLM 执行以下能力：
1. 多模态文档解析（PDF + Excel）
2. 跨文档实体对齐（识别相同工程项目）
3. 数值提取与比对
4. 差异原因分析
5. 证据定位与引用
