## 四、与 PRD 的差异说明

> 对照 PRD v0.2 中 4.3.5 节的 F0-F10 分类，标注本清单与 PRD 的差异点，供后续 PRD 修订参考。

### 4.1 PRD 分类粒度不足

| 差异点 | PRD 现状 | 本清单补充 |
|--------|----------|------------|
| 子工程级文件未展开 | PRD 仅定义 F0-F10 共 11 类项目级文件 | 本清单新增第二章，细化 7 类子工程（开关站/街坊站、10kV电缆、0.4kV电缆、通信、信息采集、PML柜、站内电缆）的文件清单 |
| 子文件类型未列举 | PRD 的 F1-F4 仅到大类（如"可研/初设报告"） | 本清单区分了报告、图纸、概算书PDF版、概算书软件版、材料清册、拆除清单等子文件 |
| 软件版概算书格式未说明 | PRD 未提及 .bpz17 / .BDD3 格式 | 本清单明确标注软件版概算书的两种格式 |

### 4.2 PRD 分类准确性修正

| 差异点 | PRD 现状 | 本清单修正 |
|--------|----------|------------|

### 4.3 PRD 未覆盖的文件类型

| 文件类型 | 说明 | 建议 |
|----------|------|------|

### 4.4 命名规范化建议

PRD 4.3.6 已识别 13 类命名不一致问题并提出应对策略。本清单第三章的命名变体速查表可直接作为系统同义词映射表的数据源，补充 PRD 未覆盖的以下变体：

- 站点命名体系差异：KT/PT 缩写 vs 全称（PRD 未提及）
- 站内电缆分隔符差异：空格 vs 连字符（PRD 未提及）
- 报告/图纸/材料清册的子目录命名变体（PRD 仅提及子工程级命名，未展开子目录级）
- 软件版概算书扩展名差异：.bpz17 vs .BDD3（PRD 未提及）
